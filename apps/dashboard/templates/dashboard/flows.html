How to Connect Google Drive

×

Step 1: Set Up Google Cloud Project

  1. Go to Google Cloud Console
  2. Create a new project (or select an existing one)
  3. Enable the Google Drive API:
    • Click "Enable APIs and Services"
    • Search for "Google Drive API"
    • Click "Enable"

Step 2: Create OAuth 2.0 Credentials

  1. In Google Cloud Console, go to "Credentials"
  2. Click "Create Credentials" â†’ "OAuth client ID"
  3. Choose "Desktop app" as application type
  4. Name it (e.g., "Flowtels Drive Integration")
  5. Download the credentials as credentials.json

Step 3: Configure Your Environment

  1. Place credentials.json in your project root directory
  2. Install required packages: 
    pip install google-api-python-client google-auth google-auth-oauthlib
  3. Add packages to requirements/base.txt: 
    google-api-python-client==2.100.0
google-auth==2.23.0
google-auth-oauthlib==1.1.0

Step 4: First-Time Authentication

  1. When you first use the connection, Google will open a browser
  2. Sign in with your Google account
  3. Grant permissions to access Google Drive
  4. The system will save a token for future use
Note: Currently, the system is using mock data. Complete these steps to enable real Google Drive integration.
